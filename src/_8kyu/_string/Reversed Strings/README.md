# Solution for problem:

[Reversed Strings](https://www.codewars.com/kata/5168bb5dfe9a00b126000018/)

## Instructions

**Description:**  
Complete the solution so that it reverses the string passed into it.

### Examples

```plaintext
'world'  =>  'dlrow'
'word'   =>  'drow'
```

```plaintext
    (solution('world'), 'dlrow');
    (solution('hello'), 'olleh');
    (solution(''), '');
    (solution('h'), 'h');
```
